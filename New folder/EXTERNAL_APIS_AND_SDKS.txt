1. REACT (v19.2.0)
================================================================================
Description:
A JavaScript library for building user interfaces with component-based 
architecture and virtual DOM for efficient rendering.

Purpose of Usage:
- Building interactive and dynamic user interfaces
- Component-based architecture for reusable UI elements
- State management and lifecycle handling
- Single Page Application (SPA) development

API Endpoints/Functions/Classes Used:
- useState() - State management in functional components
  Used in: All dashboard pages, MessagingPage, ProfilePage, SignupPage
  Files: BuyerDashboardPage.tsx, ConsultantDashboardPage.tsx, MessagingPage.tsx,
         ProfilePage.tsx, SignupPage.tsx, AccountTypePage.tsx
  
- useEffect() - Side effects and lifecycle management
  Used in: BuyerDashboardPage, ConsultantDashboardPage, MessagingPage
  Files: All major page components for data fetching and subscriptions
  
- useRef() - DOM element references and mutable values
  Used in: MessagingPage (messagesEndRef, inputRef)
  Files: MessagingPage.tsx
  
- useNavigate() - Programmatic navigation
  Used in: All pages for routing and redirects
  Files: All page components
  
- useParams() - URL parameter extraction
  Used in: MessagingPage, SubmitProposalPage, PostJobPage
  Files: MessagingPage.tsx, SubmitProposalPage.tsx, PostJobPage.tsx
  
- useLocation() - Current location and state access
  Used in: MessagingPage, MainLayout, Header
  Files: MessagingPage.tsx, MainLayout.tsx, Header.tsx

4. AXIOS (v1.13.2)
================================================================================
Description:
Promise-based HTTP client for making API requests from browser and Node.js 
with support for interceptors, automatic JSON transformation, and request/
response handling.

Purpose of Usage:
- HTTP requests to backend API
- Request/response interceptors for authentication
- Error handling and retry logic
- Automatic JSON data transformation

API Endpoints/Functions/Classes Used:
- axios.create() - HTTP client instance creation
  Used in: httpClient.ts
  
- axios.get() - GET requests
  Used in: Fetching users, jobs, proposals, consultants, messages
  Endpoints: /users/me, /jobs, /consultants, /proposals, /messages/conversations,
             /messages/:userId, /jobs/buyer/:buyerId, /proposals/buyer/:buyerId
  Files: BuyerDashboardPage.tsx, ConsultantDashboardPage.tsx, MessagingPage.tsx,
         ProfilePage.tsx, PostJobPage.tsx, AdminDashboardPage.tsx
  
- axios.post() - POST requests
  Used in: Creating jobs, proposals, messages, user registration
  Endpoints: /auth/register, /auth/login, /jobs, /proposals, /messages, /consultants
  Files: SignupPage.tsx, PostJobPage.tsx, SubmitProposalPage.tsx, MessagingPage.tsx,
         ProfilePage.tsx
  
- axios.put() - PUT requests
  Used in: Updating jobs
  Endpoints: /jobs/:id
  Files: PostJobPage.tsx
  
- axios.patch() - PATCH requests
  Used in: Partial updates (proposals, messages read status, user profile)
  Endpoints: /proposals/:id/accept, /proposals/:id/reject, /messages/:userId/read,
             /users/me, /admin/consultants/:id/verify, /admin/users/:id/ban
  Files: BuyerDashboardPage.tsx, MessagingPage.tsx, ProfilePage.tsx,
         AdminDashboardPage.tsx, AccountTypePage.tsx
  
- axios.delete() - DELETE requests
  Used in: Deleting jobs
  Endpoints: /jobs/:id
  Files: BuyerDashboardPage.tsx
  
- interceptors.request.use() - Request interceptor
  Used in: httpClient.ts for adding JWT tokens
  
- interceptors.response.use() - Response interceptor
  Used in: httpClient.ts for error handling
